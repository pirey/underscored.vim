# underscored.vim

Press `<C-G><Space>` in insert mode to swap space to underscore.

![](demo.gif)

## Installation

Install using any vim plugin manager, or with vim-plug:

    Plug 'pirey/underscored.vim'

Then run `:PlugInstall`
